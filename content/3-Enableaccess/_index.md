#### Enable access

1. Sign in to the AWS Management Console with the root account.

![Delegate access to the billing console](/images/0001.png?featherlight=false&width=90pc)

2. You can choose **[My Account](https://console.aws.amazon.com/billing/home#/account)**

   - Scroll down to **IAM User and Role Access to Billing Information**
   - Select **Edit**

![Delegate access to the billing console](/images/0002.png?featherlight=false&width=90pc)

3. In the edit interface of **IAM User and Role Access to Billing Information**

   - Select **Activate IAM Access**, activate access to the Billing and Cost Management console page
   - Select **Update**

![Delegate access to the billing console](/images/0003.png?featherlight=false&width=90pc)
